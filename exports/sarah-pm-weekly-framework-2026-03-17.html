The Core Principle

You Manage the Agents. The Agents Do the Work.

The CPO, scrum master, and product intelligence agents are the domain experts. Your job is to become an expert at directing them — asking the right questions, getting high-quality output, and ensuring the work meets the bar.

How to Work with Agents

  1. Bring in the right skill expert FIRST. Before asking an agent to do work, load the relevant specialist (CPO for strategy, database expert for architecture, etc.).
  2. Ask for a plan — don't dictate one. Tell the agent what you need, not how to do it. Let it propose the approach. You'll get a better, less biased answer.
  3. QA for 30-45 minutes. Don't accept the first draft. Read it carefully. Ask follow-up questions. Refine through dialogue.
  4. Update permanent knowledge. When you find something the agent got wrong, correct its knowledge so it doesn't repeat the mistake.
  5. Validate against reality. You bring the customer conversations, the product intuition, the "that doesn't match what I'm seeing." The agent brings structure and expertise.

Rhythm

Weekly Cadence

Monday — Kickoff

Steve sets deliverables for the week. Review scorecard from prior week. Align on priorities and agent usage expectations.

30 minutes

Daily — Async Standup

Post in Slack: what you did, what you're doing, blockers, which agents you used. Keep it brief.

5 minutes

Friday — Review

Review deliverables vs. plan. Pull scorecard data. Agent usage retrospective. Set next week's plan.

30 minutes

Ad Hoc

Escalate blockers immediately. Don't wait for Friday. Use Slack or schedule 15 min with Steve.

Hard Lines

V2 Scope Rules

No New Features

V2 scope is frozen. Complete what exists at high quality. Fix bugs, workflows, gotchas. No additions until current state is solid and the V1 to V2 migration is planned.

V1 Work Decision Filter

For any remaining V1 work, ask two questions:

  • Is this work useful for the V1 to V2 transition? If no, deprioritize.
  • Can agents do it efficiently? If yes, proceed. If no, evaluate ROI.
