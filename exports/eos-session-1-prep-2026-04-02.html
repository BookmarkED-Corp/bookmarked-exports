Weekly Steve + Raju Meeting Format

45 minutes, not 90. Earn the right to expand later. Same day, same time, no exceptions.

3 min
Check-In
One personal win, one professional win each. Set the tone.
5 min
Scorecard Review
MVS numbers. On track = checkmark. Off track = Issues List. No discussion here.
5 min
Rock Status
On track / off track per person. Off track = Issues List.
5 min
To-Do Review
Done or not done from last week. Target: 90% completion rate.
25 min
IDS — Identify, Discuss, Solve
Vote top 3 issues. Take #1. Identify root cause. Discuss (10 min max per issue). Solve = a to-do with an owner and a deadline. Repeat.
2 min
Wrap
Read back all to-dos. Cascading messages (who needs to know what). Rate the meeting 1–10.

Rules for the First 4 Weeks

  • Same day, same time — no rescheduling unless emergency
  • Meeting ends on time even if Issues List isn't empty
  • Below-8 rating becomes an issue to IDS the following week
  • No laptops during IDS (phones face-down)
