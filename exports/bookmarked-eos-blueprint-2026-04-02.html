EOS (Entrepreneurial Operating System) is a set of simple tools that align everything in the business — where we're going, who's accountable, how we measure, and how we solve problems. It's 6 components. That's it.

We're not implementing all 6 today. We're locking the ones that close the gap Raju identified: no consistent rhythm, no accountability, no way to know if commitments are being kept.

Data
How do we know we're winning?
Starting today

4
Issues
What's in the way?
We have this

An open, honest list of everything blocking progress. Not a complaint board — a solve board. Every issue gets IDS'd in the weekly L10:

I
Identify
Name the root cause in one sentence. Not the symptom.
D
Discuss
Everyone shares once. No repeating. 10 min max.
S
Solve
A to-do with an owner and a deadline. Done.
  • We have 10 issues identified. Marketing gap, publisher model, Content Scanner timeline, meeting rhythm, data hygiene, pricing, hiring, expansion, classroom libraries, Rock-setting.
  • Gap: Raju adds anything missing today. We don't solve them today — we confirm the list is real and start IDS'ing them in future L10s.
5
Process
How do we do things here?
Comes later

Document the handful of core processes that run the business — how we sell, how we onboard, how we build, how we support. Not 50-page SOPs. Just the key steps so everyone does it the same way, every time.

This is not a Day 1 problem. Get the rhythm and accountability working first. Process documentation comes after we've been running L10s consistently for 4+ weeks.

6
Traction
Are we doing what we said?
Starting today

This is the component that closes the gap Raju identified. Two tools:

Rocks — 90-Day Priorities

3–7 company priorities per quarter. Each one is owned by one person, has a clear "done" definition, and is either on track or off track. No gray area.

  • Steve: EOS Operational Rhythm — 4 consecutive L10s with Scorecard data
  • Raju: Content Scanner Architecture — architecture sprint delivered and approved
  • Jan: Pipeline Target — close $[X] in new pipeline
  • Patrick: Renewal Playbook — systematic renewal process deployed
  • Steve: Marketing Seat — function staffed
  • Steve: Publisher Pipeline — 3+ active conversations, 1+ MOU
The L10 Meeting — Weekly Pulse

Same day. Same time. Every week. Non-negotiable. This is where EOS becomes real.

3 min
Check-in — one win each
5 min
Scorecard — numbers only, no discussion. Off track → Issues List.
5 min
Rocks — on track or off track. Off track → Issues List.
5 min
To-Dos — done or not done. Target: 90%.
25 min
IDS — vote top 3, take #1, identify → discuss → solve. Repeat.
2 min
Wrap — read back to-dos. Rate 1–10. Below 8 = issue for next week.
